  • Snake.io

    Snake.io is an online multiplayer version of the classic arcade game snake. Players can control their snake, eat snacks and run into one another with the goal of growing the largest snake. Before joining the server, players can input their name and watch as it is displayed on a live leaderboard.

    Technologies Used:

    • Node.js
    • Express
    • Socket.io
    • Javascript
    • HTML5 & CSS3
    Check it out

  • BMO myApp

    The BMO myBank app was created for a business competition and is a prototype serving to demonstrate necessary improvements in the current BMO app. The app is capable of using an NFC enabled android device to read and write to an NFC data chip. These capabilities were used to create interactive posters that would launch the app and display a custom message when the user tapped their phone to a poster. App also displayed conventional information (balance, settings, contact etc).

    Technologies Used:

    • Java
    • Android Studio
    • NFC Transfer API
    • XML
    Check it out

  • Memory Path

    Memory Path is a simple game designed to challenge the user's memory. The user is shown a path and given 10 seconds to memorize it. Once the time has passed, the user must guide a ball along the same path. Mistakes lose the player lives and eventually force a complete restart.

    Technologies Used:

    • ActionScript 3
    • Adobe Animate
